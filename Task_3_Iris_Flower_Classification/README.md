# 🌸 Iris Flower Classification

## 📌 Project Overview

The Iris Flower Classification project is a Machine Learning classification task that predicts the species of an Iris flower based on its sepal and petal measurements.

The objective is to train a machine learning model capable of accurately classifying flowers into one of the following species:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

This project was completed as part of the **CodSoft Data Science Internship**.

---

## 🎯 Project Objective

To develop a robust machine learning classification model that can identify Iris flower species using morphological measurements and evaluate its performance using multiple machine learning techniques.

---

## 📂 Dataset Information

The dataset contains 150 flower samples with the following features:

### Input Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

### Target Variable

* Species

The dataset is one of the most widely used benchmark datasets in machine learning and pattern recognition.

Dataset Source:

https://raw.githubusercontent.com/uiuc-cse/data-fa14/gh-pages/data/iris.csv

## 🤖 Machine Learning Models

The following classification algorithms were evaluated:

### Logistic Regression

A simple and efficient linear classification algorithm.

### Support Vector Machine (SVM)

A powerful classifier capable of finding optimal decision boundaries.

### Random Forest Classifier

An ensemble learning technique that combines multiple decision trees for improved performance.

## 📈 Model Evaluation

The models were evaluated using:

* Accuracy Score

* Classification Report
