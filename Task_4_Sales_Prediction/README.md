# 📈 Sales Prediction Using Python

## 📌 Project Overview

Sales prediction is one of the most important applications of Data Science in business. It helps organizations estimate future sales based on historical advertising data and make data-driven decisions regarding marketing investments.

This project focuses on predicting product sales using advertising expenditure across different marketing channels such as TV, Radio, and Newspaper. Multiple machine learning models were trained and evaluated to identify the most effective model for sales forecasting.

This project was completed as part of the **CodSoft Data Science Internship**.

---

## 🎯 Project Objective

To build a machine learning model capable of predicting product sales based on advertising budgets and to analyze the impact of different advertising channels on sales performance.

---

## 📂 Dataset Information

The dataset contains advertising expenditures across three marketing channels and corresponding sales figures.

### Features

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget

### Target Variable

* Sales

Dataset Size:

* 200 Records
* 4 Columns

## 🤖 Machine Learning Models

The following regression algorithms were implemented and compared:

### Linear Regression

A statistical model used to establish a relationship between advertising budgets and sales.

### Decision Tree Regressor

A tree-based model capable of capturing non-linear relationships.

### Random Forest Regressor

An ensemble learning model that combines multiple decision trees for improved prediction performance.

---

## 📈 Model Evaluation

Models were evaluated using:

* R² Score
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

### Validation Technique

* 5-Fold Cross Validation

This ensured reliable and robust model evaluation.

---

## 🏆 Best Model Selection

All models were compared using cross-validation scores, and the best-performing model was selected for final sales forecasting.

---

## 📊 Business Insights

Key findings from the analysis include:

* TV advertising showed the strongest positive correlation with sales.
* Radio advertising significantly influenced product sales.
* Newspaper advertising had comparatively lower impact on sales performance.
* Increased investment in high-performing advertising channels can potentially improve future sales outcomes.
