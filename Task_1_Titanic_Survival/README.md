# 🚢 Titanic Survival Prediction

## 📌 Project Overview

This project aims to predict whether a passenger survived the Titanic disaster using Machine Learning techniques. The model is trained on the famous Titanic dataset and analyzes various passenger features to make survival predictions.

## 🎯 Objective

Build a classification model that predicts passenger survival based on factors such as age, gender, passenger class, fare, and family information.

## 📊 Dataset

The dataset contains information about Titanic passengers, including:

* Passenger Class (Pclass)
* Gender (Sex)
* Age
* Fare
* Embarked Port
* Family Details
* Survival Status (Target Variable)
