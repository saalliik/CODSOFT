# 🚢 Task 1: Titanic Survival Prediction

## 📖 Project Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning techniques.

The model was trained using passenger information such as age, gender, passenger class, fare, family information, and other engineered features.

This project was completed as part of the **CodSoft Data Science Internship Program**.

---

## 🚀 Core Features

### 🧹 Data Cleaning
- Handled missing values
- Age imputation using Title-based median values
- Embarked missing value treatment
- Fare preprocessing

### ⚙️ Feature Engineering
- Title Extraction (Mr, Mrs, Miss, etc.)
- Family Size Calculation
- IsAlone Feature
- Family Category Feature
- Deck Extraction from Cabin
- Age Group Classification
- Fare Band Categorization

### 🔄 Data Preprocessing
- One-Hot Encoding
- Train-Test Split
- Stratified Sampling

---

## 🤖 Machine Learning Model

### Gradient Boosting Classifier

The model was trained using:

- Gradient Boosting Classifier
- GridSearchCV Hyperparameter Tuning
- Stratified K-Fold Cross Validation

---

## 📊 Model Performance

### Results

- Accuracy: **80.45%**
- ROC-AUC Score: **0.8411**
- Cross Validation ROC-AUC: **0.8798 ± 0.0303**

These results demonstrate strong predictive performance and model stability.
