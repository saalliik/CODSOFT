# CodSoft Data Science Internship

This repository contains projects completed as part of the CodSoft Data Science Internship, focusing on machine learning, data analysis, predictive modeling, and business analytics using Python.

## Projects

### 🚢 Task 1: Titanic Survival Prediction

Machine learning model to predict passenger survival using demographic and travel-related features.

### 🎬 Task 2: Movie Rating Prediction

Regression-based model for predicting movie ratings using movie metadata and historical information.

### 🌸 Task 3: Iris Flower Classification

Classification model for identifying Iris flower species based on sepal and petal measurements.

### 📈 Task 4: Sales Prediction Using Python

Machine learning model for forecasting product sales based on advertising expenditures across TV, Radio, and Newspaper channels.
