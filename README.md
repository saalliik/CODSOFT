# 🚀 CodSoft Data Science Internship Projects

Welcome to my Data Science Internship repository completed under the **CodSoft Data Science Internship Program**.

This repository contains machine learning and data science projects demonstrating data preprocessing, exploratory data analysis, feature engineering, model building, evaluation, and visualization techniques.

---

## 📂 Projects Included

### 🚢 Task 1: Titanic Survival Prediction

Predict passenger survival using machine learning algorithms and feature engineering techniques.

**Skills Used:**

* Data Cleaning
* Feature Engineering
* Classification Models
* ROC-AUC Analysis
* Hyperparameter Tuning

---

### 🎬 Task 2: Movie Rating Prediction

Build a regression model to predict movie ratings based on features such as genre, director, actors, votes, and duration.

**Skills Used:**

* Data Preprocessing
* Exploratory Data Analysis
* Regression Models
* Model Evaluation
* Data Visualization

---

### 🌸 Task 3: Iris Flower Classification

Classify Iris flowers into different species using sepal and petal measurements.

**Skills Used:**

* Classification Algorithms
* Cross Validation
* Feature Importance Analysis
* Confusion Matrix
* Machine Learning Evaluation
